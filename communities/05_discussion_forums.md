### How Discord communities work for AI builders
- **`#showcase`** channels: share what you built → get feedback and users
- **`#help`** / **`#support`** channels: fastest way to get model-specific help
- **Model release announcements**: new weights drop in Discord first
- **Collab channels**: find co-founders, contributors, freelancers
- **Bots**: many servers have AI bots you can prompt right in chat

**URL:** https://stackoverflow.com

### Sign Up
- Free: ask and answer questions, earn reputation
- Stack Overflow for Teams: private Q&A for organizations

### Code & AI
| Feature | What it does |
|---------|-------------|
| **Tags** | `langchain`, `openai-api`, `huggingface`, `pytorch`, `transformers` |
| **Stack Overflow AI** | In-editor search and answer synthesis |

### Best practices for AI questions on SO
- Include: model name, SDK version, full error traceback, minimal reproducible example
- Tag correctly: `python` + `openai-api` + `langchain` etc.
- Check existing answers first — most API errors are already answered

## HackerNews

**URL:** https://news.ycombinator.com

### Sign Up
- Free, email only
- Karma system — upvotes unlock commenting and posting

### Role in the AI Stack
- **"Show HN"** posts: launch your AI product here for a sharp technical audience
- **"Ask HN"** posts: get real feedback on ideas and approaches
- Best for: launch announcements, model releases, research papers
- Who's here: founders, engineers, investors, researchers
- Comment quality: high signal, direct, critical

### Tips for Show HN launches
- Post on weekdays 8–11am US Eastern time for max visibility
- First comment: post your own context and what kind of feedback you want
- Respond to every comment in the first 2 hours
